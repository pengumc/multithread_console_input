Detailed Description

Brief this class holds an angle that ranges from -PI to PI.

setting an angle outside the range will automatically convert the angle to one within the range:
CAngle Phi();
phi = 3;
will result in a = 3.0

phi = 4;
will result in a = 4 - 2*PI = -2.28....
